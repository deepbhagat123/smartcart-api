# SmartCart API

A retail backend REST API built with Spring Boot and MySQL, inspired by Walmart's core functionality.

## Tech Stack
- Java 17
- Spring Boot 3.5
- MySQL 9.6
- Spring Data JPA / Hibernate
- Maven

## Setup

### Prerequisites
- Java 17
- MySQL 9.6

### Database Setup
```sql
CREATE DATABASE smartcart;
```
